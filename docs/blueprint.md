# **App Name**: EduVision Consulting

## Core Features:

- Service Listings: Categorized listings of consulting services for Bachelor, Master, PhD, and diploma programs with rich text and image support.
- Success Story Showcase: Display student success stories with image uploads, watermarking, descriptions, and social media sharing options.
- Blog Publishing: A blog platform for publishing articles with rich text editing, image uploads, and social media sharing.
- Career Application Portal: A form for users to submit CVs and personal information for job/internship applications.
- Referral Program System: System to manage user referrals, track referral codes, and calculate commissions.
- Admin Dashboard: Secure admin panel to manage services, FAQs, success stories, blog posts, job applications, and referrals.
- Firestore Integration: Utilize Firestore for storing and retrieving services details, FAQs, success stories, blog posts, job applications, referrals, and user data.

## Style Guidelines:

- Primary color: Ocean blue (#4285F4) to convey trust and professionalism.
- Background color: Light blue (#E3F2FD), subtly desaturated to ensure readability and visual comfort.
- Accent color: Gold (#FFC107) to highlight important elements and calls to action, drawing the eye without overwhelming the user.
- Headline font: 'Playfair', a modern serif to evoke elegance and professionalism; body font: 'PT Sans', a humanist sans-serif.
- Use modern, flat icons related to education, international travel, and consulting.
- Clean and structured layout with clear sections for each service, success story, and blog post.
- Subtle animations on button hovers and transitions to improve user experience.
